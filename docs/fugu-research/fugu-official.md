# Sakana Fugu Official Page
**URL:** https://sakana.ai/fugu

## Key Concepts
- **Fugu:** A multi-agent system masquerading as a single frontier model. It leverages an orchestrator to dynamically route user requests to an optimal pool of underlying models.
- **Fugu Ultra:** A deeper, fixed pool variant optimized for high-stakes domains (like cyber security, patent research, and Kaggle competitions).
- **Core Architecture:** Powered by **TRINITY** (evolved coordination) and **Conductor** (RL-trained coordination strategies).
- **Benchmarks:** Outperforms Claude Fable and GPT-4o on complex agentic benchmarks (LiveCodeBench, SWE Bench Pro, GPQA-D).

## Description
The official product page highlights how Fugu breaks vendor lock-in by dynamically swapping out models in the backend. It visualizes the TRINITY role-assignment system and showcases Fugu's ability to act as an "AI project manager that hires a team."
